# Licença Parental - Simulador Portugal

Website interativo para simular e compreender a licença parental em Portugal (opção 150+30 dias).

## Funcionalidades

- **Calendário interativo** — escolha a data prevista do parto
- **Simulador flexível** — configure quem goza primeiro, dias exclusivos da mãe/pai e partilha dos 30 dias extra
- **Timeline visual** — visualize as datas de cada período
- **Regras explicadas** — secção expansível com as regras da Segurança Social

## Desenvolvimento

```bash
npm install
npm run dev
```

Abre [http://localhost:3000](http://localhost:3000).

## Deploy na Vercel

1. Push do projeto para GitHub
2. Importe em [vercel.com](https://vercel.com)
3. Deploy automático

Ou use a CLI:

```bash
npm i -g vercel
vercel
```

## Baseado em

- Regulamento da Segurança Social (Subsídio Parental Inicial)
- PDF e Excel de referência na pasta `files/`
